Påkrevd DPI properties Denne må settes for å bruke DPI
difi.move.feature.enableDPI=true true Skrur på muligheten til å sende meldinger til innbygger
Valgfrie DPI properties Default verdi Disse kan brukes
difi.move.dpi.language NO
difi.move.dpi.priority NORMAL
difi.move.dpi.securityLevel 3 Sett sikkerhetsnivå 3 eller 4 på meldingen
difi.move.dpi.printSettings.shippingType ECONOMY Print innstilling
difi.move.dpi.printSettings.inkType BLACK_WHITE Print innstilling
difi.move.dpi.printSettings.returnType SHREDDING Print innstilling
